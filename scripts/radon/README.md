# Radon

Radon is a VSCode/Cursor extension that simplifies running React Native apps on iOS and Android devices/simulators. 

## HybridApp Setup

1. Download **Radon IDE** extension from the VSCode or Cursor extension store. 
   - [VSCode](https://marketplace.visualstudio.com/items?itemName=swmansion.react-native-ide)
   - [Cursor](https://marketplace.cursorapi.com/items/?itemName=swmansion.react-native-ide)
2. Turn off Metro if it is already running in the background.
3. Open the Radon panel from the sidebar.
4. Select **Custom configuration** and add an iOS or Android device.
5. Press **Run** to start the app.
